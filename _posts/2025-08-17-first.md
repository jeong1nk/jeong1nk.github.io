# 1.마크다운이란?
텍스트 기반의 마크업언어로 2004년 존그루버에 의해 만들어졌으며 쉽게 쓰고 읽을 수 있으며 HTML로 변환이 가능하다. 특수기호와 문자를 이용한 매우 간단한 구조의 문법을 사용하여 웹에서도 보다 빠르게 컨텐츠를 작성하고 보다 직관적으로 인식

# 2.마크다운 문법 정리
## 1. 헤더(Headers)

```
# This is a H1
## This is a H2
### This is a H3
#### This is a H4
##### This is a H5
###### This is a H6
```

<h1 class="no-toc">This is a H1</h1>
<h2 class="no-toc">This is a H2</h2>
<h3 class="no-toc">This is a H3</h3>
<h4 class="no-toc">This is a H4</h4>
<h5 class="no-toc">This is a H5</h5>
<h6 class="no-toc">This is a H6</h6>

- 6개까지만 지원한다.

## 2. 인용(BlockQuote)

```
> This is a first blockqute.
>	> This is a second blockqute.
>	>	> This is a third blockqute.
```

> This is a first blockqute.

>   > This is a second blockqute.

>   >   > This is a third blockqute.

- 안에 다른 마크다운 요소를 포함할 수 있다.

```
> 인용 안에
> - 리스트 안에
>
>   ```
>   코드
>   ```
```

> 인용 안에
> - 리스트 안에
>
>   ```
>   코드
>   ```

## 3. 목록(List)
- 번호가 있는 목록

```
1. 첫번째
2. 두번째
3. 세번째
```

1. 첫번째
2. 두번째
3. 세번째

- 번호가 없는 목록

```
* 빨강
  * 녹색
    * 파랑

+ 빨강
  + 녹색
    + 파랑

- 빨강
  - 녹색
    - 파랑
```

* 빨강
  * 녹색
    * 파랑

+ 빨강
  + 녹색
    + 파랑

- 빨강
  - 녹색
    - 파랑

<br>
- "*", "+", "-" 전부 같은 글머리를 출력한다.
- "*", "+", "-" 문자를 출력하고 싶을때는 문자 앞에 "\"를 붙이면 된다.

## 4. 코드(Code)
<h3 class="no-toc">한 줄 코드</h3>
- 한 줄은 들여쓰기로 가능하다.

```
    한 번만 들여쓰기한 코드입니다.
```
    그러면 이렇게 출력됩니다.
    
<h3 class="no-toc">코드 블럭</h3>
- 앞 뒤로 ``` 또는 ~~~ 를 사용한다.
- ``` 또는 ~~~ 의 바로 뒤에 언어를 지정해주면 syntax color가 적용된다.

~~~
```python
print("Hello World")
```
~~~

```python
print("Hello World")
```
